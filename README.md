# Webrunning-2.0-Loot-Charts
A dump of pre-generated Webrunning 2.0 Gear sorted by Rarity, Rarity + Color, and Rarity + Gear Type.

Each document has 1000 pieces of gear in it and instructions for rolling a random one using a d10 and a d100.

Go [here](https://publish.obsidian.md/xeonrapture/xr/XRRPG+2.0/Rule+Modules/Webrunning+2.0/Webrunning+2.0) for Webrunning 2.0 rules.

Go [here](https://publish.obsidian.md/xeonrapture/xr/XRRPG+2.0/Rule+Modules/Webrunning+2.0/Gear+Generation/Gear+Generation+and+Modification) for Gear Generation and Modification rules.
